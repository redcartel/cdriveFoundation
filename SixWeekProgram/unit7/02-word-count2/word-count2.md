## Word Count Revisited

Let's find the most common words in Moby Dick.

Make a dictionary where each key is a word and the value is the number of times that word appears.

Once you've made that dictionary, make a list of tuples. The tuples should be (# of appearances of the word, the word)

If you use a sort function on a list of tuples, it will sort by the first value of the tuples.
