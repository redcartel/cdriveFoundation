## Byte Academy Python Foundations

Curriculum. Each unit is one hour of lecture and two hours of exercise and assessment.

Unit 1:  The python interpreter. Python files. Python expressions. Numerical and string operations.

Unit 2:  Booleans and conditional logic. Code blocks and indentation. If statements. While loops.

Unit 3:  Sequences. Ranges and Tuples. Index and slicing. For loops.

Unit 4:  Lists and list operations. For loops and lists.

Unit 5:  Functions. The return statement. Local scope. Globals.

Unit 6:  Text file input and output. Line oriented files.

Unit 7:  Dictionaries and json files.

Unit 8:  Program structure. Designing larger programs. 

Unit 9:  Python modules. Pip. HTTP Requests. HTML document structure.

Unit 10: Webscraping with Beautifulsoup.

Unit 11: Review of concepts and discussion of where to go next 

Day  12: Three hour final assessment.
