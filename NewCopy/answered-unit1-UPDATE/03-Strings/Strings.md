## String Exercises

* Give the user the prompt "Enter some text: " and save the value with an input() statement into a variable called 'text'

* Output the text in uppercase (.upper())

* Print the text in lowercase (.lower())

* Print the text repeated 3 times

* Print out the first letter of the text (text[0])

* Print out the length of the text

* Save the boolean value of the test ```"Byte" in text``` in a variable and then print that variable.

* Save the boolean value of len(text) > 10 in a variable and print that variable.

* Check out the documentation for the built in string methods: [https://docs.python.org/3/library/stdtypes.html#string-methods] can you do anything interesting with them?
