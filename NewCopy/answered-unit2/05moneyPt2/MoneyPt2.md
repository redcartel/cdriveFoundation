## Money problem Part 2

* Let's add some features to the correct change calculator from unit 1.

* Make it so it doesn't print a line if there are 0 of that bill or coin.

* Make it so it prints the plural of a bill or coin if there are more than one and the singular if there is only one.

* For instance

```
How much money do you have? 40.81

You have 2 twenty dollar bills.
You have 3 quarters.
You have 1 nickel.
You have 1 penny.
```
